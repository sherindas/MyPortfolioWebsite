My Journey in Building This Portfolio

I started by sketching a simple wireframe for my portfolio design. My main goals were: a clean layout, responsive design, and easy navigation.

I built the site using HTML, CSS, and JavaScript. For styling, I made use of CSS Grid and Flexbox to make sure the design adapts across devices.

One of the biggest challenges was making the portfolio section interactive. I solved this by adding hover effects with CSS transitions.

Finally, I deployed the website on GitHub Pages, which makes it free and fast to share with recruiters and friends.
